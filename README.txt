Schnellstart:
1. Erstelle auf GitHub ein neues öffentliches Repository, z. B. "ki-clipper-site".
2. Lade den GESAMTEN Inhalt dieses Ordners in das Repository hoch.
3. Öffne im Repository: Settings → Pages.
4. Wähle unter "Build and deployment":
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
5. Speichern und einige Minuten warten.
6. GitHub zeigt anschließend die öffentliche URL an.
